# Student Management System (C Language)

This project is a simple C program to store and display student details such as:
- Aadhar Number
- APAR ID
- Email ID
- Phone Number

## Features
- Uses structure in C
- Displays student details
- Beginner friendly

## How to Run
1. Compile the program using gcc
   gcc student_management.c
2. Run the program
   ./a.out
